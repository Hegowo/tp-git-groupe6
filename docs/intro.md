# Introduction

Ceci est une documentation simple. Vous devez compléter ce fichier dans un ticket.

Git est un logiciel de gestion de versions décentralisé. À l'origine, il a été développé en 2005 par Linus Torvalds, le créateur bien connu du noyau du système d'exploitation Linux.

- Le contrôle de version permet de suivre les modifications apportées au code source d'un logiciel.
- Une meilleure autonomie des équipes
- Une visualisation de l'historique de vos projets
